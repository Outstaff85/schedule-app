# Обновление дизайна — Outstaff85/schedule-app

Эти файлы заменяют одноимённые в репозитории. Всё, что внутри, — рабочий код
(localStorage, Google Sheets, GPS, экспорт сохранены), изменён только дизайн.

## Что нового
- 🎨 Новая иконка приложения (полоски смен + курсор «сейчас») — `icon-192.png`, `icon-512.png`
- 📊 Вкладки перенесены в верхнюю панель (иконка над подписью): Today / Schedule / Clients / Admin
- 📅 Гантт: полосы смен заполняют ячейку целиком (07:00 / 15:00), «OFF» для выходных, синие — через полночь
- 🗓 Календарные даты под днями недели + ивритские буквы
- 💰 Израильский расчёт зарплаты: обычные ≤8ч ×1.0, сверхурочные +2ч ×1.25 затем ×1.5, Шаббат (суббота) база ×1.5
- 💵 Тарифы по каждому работнику (поле в карточке работника)

## Изменённые файлы
| Файл | Что |
|---|---|
| `schedule-manager.html` | Приложение менеджера — весь новый дизайн |
| `worker-punch.html` | Приложение пунч-часов (без изменений кода, подхватывает новую иконку) |
| `icon-192.png` | Новая иконка PWA 192×192 |
| `icon-512.png` | Новая иконка PWA 512×512 |

> Манифесты (`manifest.json`, `manifest-punch.json`) и Service Worker не менялись — не трогайте их.

## Как залить
**Вариант A — через сайт GitHub (проще):**
1. Откройте https://github.com/Outstaff85/schedule-app
2. «Add file» → «Upload files»
3. Перетащите все 4 файла из этой папки
4. Внизу впишите сообщение коммита (например: `Redesign: top-nav tabs, new icon, Israeli salary`)
5. «Commit changes»

**Вариант B — через git локально:**
```bash
# скопируйте 4 файла в корень локальной копии репо, затем:
git add schedule-manager.html worker-punch.html icon-192.png icon-512.png
git commit -m "Redesign: top-nav tabs, new icon, fill-cell Gantt, Israeli salary"
git push
```
